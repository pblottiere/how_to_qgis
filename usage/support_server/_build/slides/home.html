Diverses distributions GNU/Linux supportées:

Installation (2)


Nécessite:
  • un serveur web (NGINX, Apache, …)
  • en environnement graphique (serveur X)
http://localhost/qgisserver?
MAP=/home/user/qgis-server-tutorial-data-master/world.qgs&
LAYERS=countries&
SERVICE=WMS&
REQUEST=GetMap&
CRS=EPSG:4326&
WIDTH=400&
HEIGHT=200
