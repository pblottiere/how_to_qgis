Services (2)

WMS

Requêtes:

  • norme OGC: GetCapabilities, GetMap, GetLegendGraphics, …
  • custom QGIS: GetPrint, GetProjectSettings, …

Paramètres:

  • norme OGC: MAP, LAYERS, BBOX, CRS/SRS, …
  • custom QGIS: SHOWFEATURECOUNT, HIGHLIGHT_GEOM, …

Clients (3)

WEB: custom

  • flux WMS affichée dans une page HTML classique
  • OpenLayers, Leaflet, …

var map=new OpenLayers.Map(’map’);
var layer=new OpenLayers.Layer.WMS(
  "Glacier fronts1",
  "http://qgis.sequanux.org/cgi-bin/project/landsat/qgis_mapserv.fcgi",
  {layers: "fronts4", transparent:"true", format:"image/png"},
  {isBaseLayer: false}
);
map.setCenter(new OpenLayers.LonLat(12,79),9);
map.addLayer(layer);
